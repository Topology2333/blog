```md
| 命令   | 含义                  |
| ------ | --------------------- |
| x      | 删除当前光标所在字符  |
| A      | 直接在行末添加        |
| d      | 删除命令，后跟对象    |
| dd     | 删除整行              |
| ndd    | 删除 n 行             |
| U      | 恢复改行状态/撤销恢复 |
| CTRL-R | 重做撤销              |
| p      | 粘贴（从上方）        |
| r      | 替换单个字符          |
```

```md
| 对象/光标移动 | 含义                                              |
| ------------- | ------------------------------------------------- |
| nw            | 向前移动 n 个单词，不包括它的第一个字母           |
| ne            | 光标到当前计起的第 n 个单词末尾，包括最后一个字母 |
| $             | 光标到行末                                        |
| 0             | 光标到行首                                        |
```
